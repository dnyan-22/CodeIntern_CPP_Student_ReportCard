# Student Report Card Generator (C++)

## Description
This project is a C++ program that generates student report cards.
It stores student details, calculates total marks, percentage, and grade.

## Features
- Stores student name, roll number, and marks
- Calculates total and percentage
- Assigns grade based on percentage
- Uses Object-Oriented Programming (OOP)

## Concepts Used
- Classes and Objects
- Functions
- Arrays
- Conditional statements

## Platform
- Online C++ Compiler (Programiz)
